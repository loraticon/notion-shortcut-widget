노션 단축키 치트시트

작업 효율을 극대화하는 필수 단축키 모음

1. 텍스트 서식 단축키

굵게 (Bold)
Ctrl/Cmd + B
기울임 (Italic)
Ctrl/Cmd + I
밑줄 (Underline)
Ctrl/Cmd + U
취소선 (Strikethrough)
Ctrl/Cmd + Shift + S
인라인 코드 (Code)
Ctrl/Cmd + E
웹 링크 추가
텍스트 드래그 후 Ctrl/Cmd + K

2. 마법의 마크다운 블록

체크박스 (To-do)
[] + 스페이스바
글머리 기호 (Bullet)
- 또는 * + 스페이스바
번호 매기기 (Number)
1. + 스페이스바
토글 목록 (Toggle)
> + 스페이스바
인용구 (Quote)
" + 스페이스바
제목 1, 2, 3 (Heading)
# , ## , ### + 스페이스바

3. 페이지 편집 단축키

새 페이지 만들기 앱에서만 동작합니다. 브라우저는 X
Ctrl/Cmd + N
새 탭 열기 앱에서만 동작합니다. 브라우저는 X
Ctrl/Cmd + T
다크/라이트모드 전환 밤에 눈이 피로할 때 최고예요! 👀
Ctrl/Cmd + Shift + L
블록 위/아래로 이동 마우스 드래그보다 훨씬 정확하고 편합니다
Ctrl/Cmd + Shift + 방향키(↑/↓)
